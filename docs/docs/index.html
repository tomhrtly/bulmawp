Getting Started

Installation

Installing BulmaWP onto your WordPress website is super easy and fast. Just follow the instructions below to get started.

Download the latest version of BulmaWP by clicking the Download button at the top of the page.

Go to to Appearance > Themes within your WordPress dashboard.

Click "Add New" at the top of the page.

Click "Upload Theme" at the top of the page.

Choose the .zip file from where you downloaded it onto your computer.

Click "Install Now".

Click "Activate".

Child Theme

Since BulmaWP is a starter theme with WordPress functionality built right in, you don't need to create a child theme and instead you can start modifying the theme to what you need.

But in the case you want to make a child theme, then just follow the steps below. Or, if you're feeling lazy, just download our Child Theme.

Create a new folder called "bulmawp-child" within the themes directory of your WordPress installation.

Create a file called "style.css" and add the following code. 

/*
Theme Name: BulmaWP Child Theme
Theme URI: https://bulmawp.io/
Author:
Author URI:
Description: BulmaWP Child Theme
Version: 1.0
Template: bulmawp
*/

Create a file called "functions.php" and add the following code. 

<?php
function bulmawp_child_enqueue() {
  $parent_style = 'bulmawp';
  wp_enqueue_style( $parent_style, get_template_directory_uri() . '/style.css' );
  wp_enqueue_style( 'child-style',
    get_stylesheet_directory_uri() . '/style.css',
    array( $parent_style ),
    wp_get_theme()->get('Version')
  );
}
add_action( 'wp_enqueue_scripts', 'bulmawp_child_enqueue', 11 );
?>

Go to to Appearance > Themes within your WordPress dashboard.

Click "Activate" under BulmaWP Child Theme.
